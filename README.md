## Project Set-up

```bash
$ git clone https://github.com/Govindsavaliya/unitask-practical-task.git
```

## Installation

```bash
$ npm i -g @nestjs/cli
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Opening the swagger

```bash
$ http://localhost:8080/api-doc in browser
```
