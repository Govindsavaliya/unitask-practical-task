## Project Set-up

```bash
$ git clone https://github.com/Govindsavaliya/unitask-practical-task.git
```

## Installation

```bash
$ npm i -g @nestjs/cli
$ npm install
```

## Running the app

```bash
# watch mode
$ npm run start:dev
```

## Opening the swagger

```bash
# in browser 
$ http://localhost:8080/api-doc 
```
